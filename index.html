C'est Quoi HTML?

(Hyper Text Markup Language) C'est un language qui permet de décrire tout ce qui s'affiche dans l'écran de ton navigateur.

Création de fichier HTML

HTML fichier pour le créer un simple éditeur de texte est suffisant mais si tu utilises un éditeur avancé comme VsCode reste mieux pour que tu profiteras de la coloration syntaxique et aidra beaucoup avec les grand fichiers. 
<tag attribut="valeur">

Structure d'un page html

le premiére ligne dans le fichier HTML sera toujours une balise unique indiquant que notre fichier et de type HTML 
<!DOCTYPE html>
puis notre document sera encadrer par les balises html 
<html>...</html>
à l'intérieur on trouvera alors un en-téte et un corps de fichier. l'en-téte est encadrés par la balise head 
<head>...</head>
et le corps par la balise body 
<body>...</body>
dans l'en-téte on trouve toutes les métadonnées de notre site comme le titre par exemple. dans le corps c'est là ou nous allons écrire tous le contenu de notre page web, pour cela on dispose nombreuse balises

Les balise

Une balise est un mot clé entourée des signe < et > quand on veut ajouter une information dans une page HTML on l'encadre entre une balise ouvrante et une balise fermente, par exemple: 
<p> informations </p>
Parfois une balise pourra comporter des attributs sous la forme de clé valeur, par exemple: 
<a href="www.google.com><a>

Les attributs

Attribut Description
action L'URI d'un programme qui traite les informations envoyées par le formulaire.
align Cet attribut définit l'alignement horizontal de l'élément.
alt Un texte alternatif à afficher lorsque l'élément ne peut pas être affiché.
autofocus Cet attribut indique que l'élément doit recevoir le focus automatiquement une fois que la page est chargée.
autoplay Cet attribut indique que l'élément audio ou vidéo doit être lancé dès que possible.
charset Cet attribut déclare l'encodage de caractères utilisé pour la page ou le script.
checked Cet attribut indique si l'élément doit être vérifié.
class Cet attribut permet de définir la ou les classes auxquelles appartient un élément afin de le manipuler en script ou de le mettre en forme avec CSS.
colspan Cet attribut définit le nombre de colonnes sur lequel une cellule dans le tableau doit s'étendre.
controls Cet attribut indique si le navigateur doit afficher les contrôles de lecture du média pour l'utilisateur.
disabled Cet attribut indique si l'utilisateur peut interagir avec l'élément.
for Cet attribut décrit l'élément auquel se rapporte l'élément courant.
height cet attribut définit la hauteur occupée par l'élément.
hidden Cet attribut empêche le rendu d'un élément sur une page tout en conservant les éléments fils actifs.
href L'URL de la ressource liée.
id Cet attribut permet d'identifier un élément d'un document de façon unique.
lang Cet attribut définit la langue utilisée dans l'élément.
list Cet attribut constitue une liste d'options prédéfinie qui est suggérée à l'utilisateur.
loop Cet attribut indique si le média courant doit recommencer au début une fois que sa lecture est terminée.
max Cet attribut indique la valeur maximale autorisée.
maxlength Cet attribut définit le nombre maximal de caractères autorisé dans l'élément.
minlength Cet attribut définit le nombre minimal de caractères qui doivent être saisis dans l'élément.
method Cet attribut définit la méthode HTTP à utiliser pour l'envoi des données du formulaire (GET par défaut ou POST
min Cet attribut indique la valeur minimale autorisée.
muted Cet attribut indique si le son de la vidéo doit être coupé lorsque la page est chargée initialement.
name Le nom de l'élément qui peut être utilisé par le serveur pour identifier le champ d'un formulaire.
novalidate Cet attribut indique que les données du formulaire ne doivent pas être validées lors de l'envoi.
placeholder Cet attribut fournit une indication à l'utilisateur sur ce qu'il peut saisir dans le champ.
poster Cet attribut est une URL qui indique l'image à afficher tant que l'utilisateur n'a pas lancé la vidéo ou déplacé le curseur.
readonly Cet attribut indique si l'élément peut être édité.
rel Cet attribut définit la relation entre l'objet cible et l'objet du lien.
required Cet attribut indique si l'élément doit obligatoirement être renseigné dans le formulaire.
rowspan Cet attribut définit le nombre de lignes sur lesquelles une cellule doit s'étendre.
selected Cet attribut définit la valeur qui sera sélectionnée au chargement de la page.
src Cet attribut indique l'URL du contenu embarqué.
style Cet attribut définit des styles CSS qui auront la priorité sur ceux définis précédemment. Il ne devrait être utilisé qu'à des fins de tests, car il est conseillé d'utiliser un/des fichier(s) à part pour gérer la mise en forme.
target
title Cet attribut définit un texte expliquant le contenu de l'élément et qui est généralement affiché sous la forme d'une bulle d'information au survol de l'élément.
type Cet attribut définit le type de l'élément.
width Cet attribut définit la largeur occupée sur l'écran.
